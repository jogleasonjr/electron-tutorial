Leveraging Angular in your App

This tutorial demonstrates how to integrate Angular but the idea is to show how any 3rd party library can be leveraged similar to how it might be done in a normal web application.

Integrating Angular

First we need to get angular from npm.

$ npm install angular

Next we need to add references to angular scripts in our html.

index.html
<!DOCTYPE html>
<html ng-app="myapp">
  <head>
    <meta charset="UTF-8">
    <title>Hello World!</title>
    <script src="node_modules/angular/angular.js"></script>
    <script src="index.js"></script>
  </head>
  <body ng-controller="VersionController">
    <h1>Hello World!</h1>
    We are using Node.js {{nodeVersion}}
    and Electron {{electronVersion}}.
  </body>
</html>

Next we add the script that initializes angular.

$ touch index.js
index.js
angular
  .module('myapp', [])
  .controller('VersionController', function ($scope) {
    $scope.nodeVersion = process.version;
    $scope.electronVersion = process.versions['electron'];
  });

Run once again to see the app running through Angular.

$ electron .
