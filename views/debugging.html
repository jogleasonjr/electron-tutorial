Debugging

In electron you have multiple processes to debug, you have access to both node debuggers and chromium debuggers.

debugging node, the main proces.
$ electron --debug-brk .

Then in another shell process

$ node debug localhost:5858
debug> c

See node debugger api for full reference

Debugging browser windows, the render processes.

These processes can be attached to via apis on the browser windows themselves. The debugger is the standard chromium debugger panel.

By default the toggle dev tools commands are bound to the main menu and keyboard shortcut: CmdOrCtrl+Shift+I

methods
  • webContents.openDevTools([options])
  • webContents.closeDevTools()
  • webContents.isDevToolsOpened()
  • webContents.toggleDevTools()
events
  • 'devtools-opened'
  • 'devtools-closed'
  • 'devtools-focused'
