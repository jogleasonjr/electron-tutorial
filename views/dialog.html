dialog module

A main process api for showing native system dialogs, such as opening files or alerting, so web applications can deliver the same user experience as native applications.

example
var dialog = require('dialog');
dialog.showOpenDialog({
  title: 'Open Text Files',
  properties: ['openFile', 'multiSelections'],
  filters: [
    { name: 'Text Files', extensions: ['txt'] },
    { name: 'All Files', extensions: ['*'] }
  ]
}, function (files) {
  dialog.showMessageBox({
    type: 'info',
    title: 'Selected Files',
    buttons: ['Ok'],
    message: 'The files you selected were:',
    detail: files.join('\n')
  });
});
Methods
  • dialog.showOpenDialog([browserWindow][, options][, callback])
  • dialog.showSaveDialog([browserWindow][, options][, callback])
  • dialog.showMessageBox([browserWindow][, options][, callback])
  • dialog.showErrorBox(title, content)
