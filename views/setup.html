Electron Setup dependencies

Git-scm

http://www.git-scm.com/

Highly recommended. Necessary for submitting patches and fixes and working with electron and node.js code. It is also an excellent way to get a good bash shell on Windows.

node.js

http://nodejs.org

This is needed to get and drive electron, as well as obtain modules and drive tools and tests.

python 2.7 (specifically this version)

http://python.org

This is needed to build native addons on all platforms.

Visual Studio (community edition is free)

https://www.visualstudio.com

This is needed to build native modules on Windows only.
